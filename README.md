# Nanut Lertmahakiat Portfolio

Personal portfolio website for Healthcare IT, Technical Project Lead, System Integration, and Digital Health opportunities.

Live site:
`https://portfolio-nanutlmhk.vercel.app/`

GitHub repo:
`https://github.com/nanutlmhk/portfolio`

## Overview

This portfolio is designed to present experience in:

- Healthcare IT leadership
- System integration and interoperability
- Hospital workflow and clinical systems
- Medical device connectivity
- Technical project delivery
- Support, troubleshooting, and implementation

The site supports both English and Thai, includes light/dark mode, and is optimized for deployment on Vercel.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS

## Main Sections

- Home
- About
- Experience
- Project
- Contact

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run lint:

```bash
npm run lint
```

Preview production build locally:

```bash
npm run preview
```

## Deployment

This project is ready for deployment on Vercel.

Recommended Vercel settings:

- Framework preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`

## Content Notes

Most portfolio content is managed from:

- `src/data/portfolioData.ts`

This makes it easier to update:

- English and Thai content
- Home section messaging
- Hospital experience entries
- Project details
- Contact information

## Assets

Static assets used by the portfolio are stored under:

- `public/assets/`

This includes:

- Resume PDF
- Hospital experience image
